Detailed Description

The class for a GENE block.

The contents are identical to that of a .gen file
This class doesn't have any useful functions yet - I don't actually know how to decode genetics at the moment.

If $prayfile is not null, all the data for this block will be read from the PRAYFile.

Parameters:
$prayfileThe PRAYFile object this block belongs to. Can be null.
$nameThe block's name. This is a creature's moniker with .genetics appended.
$contentThe block's binary data. Used when constructing from a PrayFile
$flagsThe block's flags, which apply to the binary data as-is.

Prefer using IsFlagSet. (This may be deprecated/removed in future releases) Least Significant Bit determines whether the block is compressed (1) or not. All other bits are 0 in all c2e-compatible PRAY blocks.
